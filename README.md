### xiaoming-minecraft
# 小明-MC-QQ 互通插件
该插件为 `Bukkit`、`Spigot`、`Paper` 等 `Minecraft` 服务器提供关联 QQ 的支持，使其能在 QQ 上进行包括但不限于执行控制台指令、分频道的互相聊天、申请执行指令、跨区域 @ 用户等操作。
