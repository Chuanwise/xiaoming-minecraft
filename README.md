## 服务器频道消息格式
### 可用变量
|变量名|含义|
|---|---|
|`channel.name`|频道名|
|`channel.head`|频道消息标志|
|`channel.serverTag`|频道关联服务器标记|
|`channel.worldTag`|频道关联世界标记|
|`sender.code`|发送方QQ|
|`sender.name`|发送方QQ名|
|`sender.alias`|发送方备注或QQ名|
|`sender.id`|发送方绑定的 ID，可能为`null`|
|`group.code`|发送者所在群号|
|`group.name`|发送者所在群名|
|`group.alias`|发送者所在群备注或群名|
|`message`|消息|
|`time`|当前时间|
